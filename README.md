# React + Zustand Online Store

A **frontend-only online store** built with **React** for the UI and **Zustand** for state management. 
This project demonstrates modern React patterns, global state management, and e-commerce functionality, including a shopping cart and responsive design — all without a backend.

## Features
- Product catalog with mock data
- Shopping cart with:
  - Add/remove items
  - Quantity management
  - Total price calculation
- Responsive design for desktop and mobile
- Login/signup popup (frontend-only)
- Modular, reusable React components
- State management with **Zustand** for simplicity and scalability

## Tech Stack

- **React** – Frontend UI library 
- **Zustand** – State management for global state (cart, modals) 
- **CSS** – Styling and animations
